# Rokubi Tattoo Studio by kusssarigawa

**Ссылка на демо:** [https://gilded-piroshki-fd7552.netlify.app/](https://gilded-piroshki-fd7552.netlify.app/)


## О проекте

Rokubi Tattoo Studio — это веб-сайт, разработанный для тату-студии, расположенной в Харькове. Проект направлен на предоставление информации о студии, услугах, мастерах и портфолио работ.

## Текущий статус

⚠️ **Проект находится на стадии разработки.** Некоторые функции и разделы могут быть недоступны или работать некорректно. Мы активно работаем над улучшением и добавлением нового функционала.

## Функционал

- **Главная страница:** краткая информация о студии и приветственное сообщение.
- **О нас:** подробная информация о студии, ее истории и миссии.
- **Мастера:** профили тату-мастеров с их биографией и примерами работ.
- **Услуги:** список предлагаемых услуг с описанием и ценами.
- **Портфолио:** галерея выполненных работ.
- **Контакты:** информация для связи и форма обратной связи.

## Технологии

- **Фронтенд:** HTML, CSS, JavaScript, React, Taiwind , TS
- **Хостинг:** Netlify

## Установка и запуск

1. Клонируйте репозиторий:

   ```bash
  git clone https://github.com/your-username/vite-react-typescript-starter.git
   ```

2. Перейдите в директорию проекта:

   ```bash
  cd vite-react-typescript-starter
   ```

3. Установите зависимости:

   ```bash
  npm install

   ```

4. Запустите приложение:

   ```bash
   npm run dev

   ```

   Приложение будет доступно по адресу `http://localhost:3000`.
